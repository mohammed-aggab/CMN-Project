GNU FRANCE

GuixSD

GuixSD et la distribution officielle du systeme GNU.
A la base elle aurait du s'appeler GNU
mais pour eviter de
creer un conflit avec les autres distributions Richard Stallman
proposa de l'appeler GuixSD.

C'est une distribution non fonctionelle;
c'est a dire que les anciennes instances ne seront pas
supprime lors d'une mise a niveau du systeme,
permettant de revenir en arriere
si la mise a niveau d'une application aurait echoue.

GuixSD

Richard Stallman

Fondateur de la FREE SOFTWARE FOUNDATION,
il developpa le systeme GNU base sur Unix en partant de zero,
en utilisant d'autre outils qu'il a concu
tels que le compilateur C GCC,
l'outils de debogage GDB,
l'editeur EMACS, l'extension de langage Guile,
la licence GPL ainsi que le noyau HURD.

Stallman GNU GCC

HURD

Le noyau hurd qui aurait du etre integre dans le systeme GNU
a eu enormement de retard dans
son developpement et il a fini par etre remplace
par le noyau Linux.

Le noyau Hurd contrairement a Linux
qui est monolithique.

c'est à dire que les fonctions fondamentals
ainsi que les pilotes ne sont
q'une seul "entite" alors que le noyau Hurd est un micro noyau.

Il fournits les fonctions essentiels.
Les autres applications doivent etre installer en general
par l'utilisateur(pilote). cela fait du noyau Hurd
une application plus modulaire que les noyaux dit monolithique.

HURD

Linux-Libre

Le noyau Linux-libre est un fork du noyau
developpe, par Linus Torvald.

Cette variante est nee d'une difference pholosophique
entre la FSF et l'Open source.

La seul difference est que ce noyau
na que des firmwares et pilotes libres

linux-libre

Parabola

Parabola est une distribution GNU/Linux basee sur
Arch linux, je dirai meme identique qui met l'accent
sur les logiciel dernier cris.

Une difference avec arch linux est quelle est constituee
exclusivement de logiciels libres.

C'est une distribution rolling release.
Cela veut dire quels est en mise a jour continue,
quelle na pas de version en intervalle comme par exemple Ubuntu.

Parabola

Trisquel

Trisquel GNU/Linux est une basee sur Ubuntu.

contrairement a Ubuntu qui a des versions a court ou long terme,
Trisquel na que des version long terme.

La deuxieme difference est quelle est constituee
exclusivement de logiciels libres.

Trisquel

GNU Emacs

Gnu emacs est un editeur puissant qui permet d'etre
personnalisable grace a l'interpreteur LIPS.

Vous pouvez en faire un IDE
,gerer vos projet avec Projectile,
configurer votre github grace a l'extension Magit,
envoyer des mails avec gnus,ces couleurs,etc.

Mais ca reste un editeur de texte assez complique à maitriser.

Emacs incarne la philosophie du logiciel libre
du fait que l'utilisateur a une liberte total sur le code source.

GNU Emacs

GIMP

Gimp est un logiciel de creation/retouche d'image.
Contrairement a son concurrent Adobe Photoshop
qui est payant est privateur,
Gimp est quand a lui libre est gratuit.
Libre veut dire que vous pouvez telecharger le code source,
le modifier et le partager.

GIMP
